# Sistema Básico de Gestión de Garaje

Aplicación de escritorio desarrollada en Python utilizando Tkinter para registrar vehículos en un garaje.

## Funcionalidades

- Registrar vehículos
- Mostrar lista de vehículos
- Limpiar campos del formulario

## Tecnologías utilizadas

- Python
- Tkinter
- Programación Orientada a Objetos

## Estructura del proyecto

garaje_app/
│
├── main.py
├── modelos/
│   └── vehiculo.py
├── servicios/
│   └── garaje_servicio.py
└── ui/
    └── app_tkinter.py

## Cómo ejecutar

```bash
python main.py
```
